## Installation

### Vue 3

```shell
# npm
$ npm install bottom-navigation-vue
# yarn
$ yarn add bottom-navigation-vue
```

### Vue 2

```shell
# npm
$ npm install bottom-navigation-vue@2.6.0
# yarn
$ yarn add bottom-navigation-vue@2.6.0
```
